# Python Craft Automation – 6 Real-World Projects

6 powerful Python automation tools made for handmade craft businesses (perfect for portfolio & interviews)

Live Demo & Source → https://github.com/faizankhan07867/python-craft-automation-6-projects

## 6 Projects (All Working – Just Run `python main.py`)

| # | Project                        | What It Does                               | Folder Link |
|---|--------------------------------|---------------------------------------------|-----------|
| 1 | Bulk File Renamer              | Rename 100s of product images in 2 seconds | [`01-bulk-file-renamer`](01-bulk-file-renamer) |
| 2 | Description Generator          | Auto-create unique product descriptions     | [`02-description-generator`](02-description-generator) |
| 3 | Pricing Calculator             | Calculate selling price with GST & discount | [`03-pricing-calculator`](03-pricing-calculator) |
| 4 | CSV Data Parser & Report       | Read CSV → Generate inventory report        | [`04-csv-data-parser`](04-csv-data-parser) |
| 5 | Trending Hashtag Fetcher       | Get Instagram-ready hashtags instantly      | [`05-hashtag-fetcher`](05-hashtag-fetcher) |
| 6 | GUI Order Tracker (Tkinter)    | Beautiful desktop app to track orders       | [`06-gui-order-tracker`](06-gui-order-tracker) |

## Tech Stack
- Python 3.8+
- Standard Library only (no pip install needed)
- Tkinter (built-in)
- 100% offline working

## How to Run (2 seconds)

# Go to each folder and run
python main.py
